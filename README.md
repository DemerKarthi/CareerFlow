# CareerFlow

CareerFlow is a personal Job Search CRM where users can manage:
- Companies
- Job Applications
- Recruiters
- Interviews
- Follow-ups
- Analytics
- Notes
- Resume Versions
- User Profile

## Tech Stack
**Frontend:** React, Vite, Tailwind CSS, React Router, Axios, React Hook Form, TanStack Query, Zod, Framer Motion
**Backend:** Node.js, Express, Sequelize, MySQL2, Multer, Nodemailer, JWT, bcrypt
**Language:** JavaScript
